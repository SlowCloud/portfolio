## Cloud & Server

- Docker :: ⬛⬛⬛⬜⬜
    - 프로세스를 격리하여 실행하기 위해 도커를 활용하고 있습니다.
    - 도커의 기본적인 작동방식을 알고 있고, CLI 작업을 진행할 수 있습니다.
    - 멀티 스테이지 빌드를 통해, 경량 컨테이너를 빌드하도록 할 수 있습니다.
- Jenkins :: ⬛⬛⬜⬜⬜
    - 개발에 집중할 수 있도록, CI/CD를 구성하기 위해 익혔습니다.
    - 파이프라인을 작성하여 간단한 CI/CD 작업을 수행할 수 있습니다.
- Kubernetes :: ⬛⬛⬜⬜⬜
    - 컨테이너 오케스트레이션을 위해 쿠버네티스를 사용했습니다.
    - 쿠버네티스의 전체적인 구조 및 작동방식을 이해하고 있습니다.
- Argo CD :: ⬛⬛⬜⬜⬜
    - 쿠버네티스 환경에서의 배포 자동화 구성 및 상태 확인을 위해 익혔습니다.
    - 간단한 배포 자동화 작업을 수행할 수 있습니다.
- Elasticsearch :: ⬛⬜⬜⬜⬜
    - MySQL 내 데이터를 검색하기 위해 활용했습니다.
    - Logstash를 활용하여 MySQL 내 데이터를 엘라스틱서치로 옮긴 경험이 있습니다.
    - 간단한 엘라스틱서치 쿼리를 작성하여 검색을 진행할 수 있습니다.
- Nginx :: ⬛⬜⬜⬜⬜
    - 리버스 프록시를 구성하기 위해 익혔습니다.
    - 간단한 리버스 프록시 작업을 수행할 수 있습니다.
- Kafka :: ⬜⬜⬜⬜⬜
- Zookeeper :: ⬜⬜⬜⬜⬜
